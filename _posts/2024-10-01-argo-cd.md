회사에서 SRE 선생님께 건의 드리기 전에 좀 공부해야할 필요성이 있을 것 같아서 집 맥미니에 하나 띄움


private 저장소인 경우 로그인 ㄱㄱ

```javascript
argocd repo add https://github.com/inha18minseokkim/IDC-MSA-Replica.git --username inha18minseokkim --password <GITHUB_PAT>
```

default namespace에 openapi-servlet 앱 배포 ㄱㄱ

```javascript
argocd app create openapi-servlet --repo https://github.com/inha18minseokkim/IDC-MSA-Replica --path common/openapi-servlet --dest-server https://kubernetes.default.svc --dest-namespace default
```
