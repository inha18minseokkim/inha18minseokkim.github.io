```java
class Muyaho {
    public Muyaho(int a) {this.a = a;}
    int a;
    public int getA() {return a;}
    public Muyaho() {a = 0;}
}
@SpringBootApplication
public class TestApplication {

    public static void main(String[] args)  {
        Function<Integer,Muyaho> a =  Muyaho::new; //input이 int인 생성자로 추정가능
        Supplier<Muyaho> b = Muyaho::new;//input이 없는 빈 생성자(여기서는 밑에 출력으로 구별을 위해 비어있진 않음)로 추정가능
				Consumer<Integer> c = System.out::println; //숫자 받아서 출력하는것으로 추정가능

        Muyaho aa = a.apply(1);
        Muyaho bb = b.get();
        System.out.println(aa.getA()); //1
        System.out.println(bb.getA()); //0
    }
}
```

이런식으로 input, output type에 따라 함수형 인터페이스 추정가능
