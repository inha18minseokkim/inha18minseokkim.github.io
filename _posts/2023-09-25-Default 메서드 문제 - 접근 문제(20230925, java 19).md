```java
interface MuyahoInterface {
    default public void hello () {
        System.out.println("ASDF");
    }
}
class MuyahoClass {
    private void hello() {
        System.out.println("SSSS");
    }
}
class TestClass extends MuyahoClass implements MuyahoInterface {

}

@SpringBootApplication
public class TestApplication {
    public static void main(String[] args) throws InterruptedException {
        TestClass testClass = new TestClass();
        testClass.hello();
    }
}
```

당연히 interface 보다는 class 가 우선하기 때문에 class 의 hello를 인식 → 근데 hello가 private임 → 그럼 당연히 interface의 default로 가야지
⇒ 가 아니라 진짜 그냥 class의 private 메서드 접근하다가 에러뱉고 끝남(? 뭐지)
