재무제표 api 호출 시 보통주 지분증권이 아닌 종목의 경우(우선주,ETF,ETN리츠 등) 종목코드 컴파일 에러, 해당 재무정보 안보여줌
해결 : 
[https://api.deepsearch.com/company/v1/financial_summary?symbol_krx=KRX:004255](https://api.deepsearch.com/company/v1/financial_summary?symbol_krx=KRX:004255)
신규 api 개발, 해당 compute 방식은 수정대응 어려움
BM에 사항 의뢰 필요
  - 우선주 상세보기로 들어왔을 때,
  1. 모회사 상세를 보여줌(가격은 그대로, 뒷자리를 0으로 ex 00088K → 000880)
  2. 그냥 안보여주고 가격만 보여줌
