해외주식 정보를 찾다 보면 티커 뒤에 알파벳이 하나 더 붙어 있는 경우가 많다. 예를 들어 `BRK.A`, `BRK.B` (버크셔 해서웨이 주식 클래스), `HOFVW` (워런트) 같은 식이다.

이 suffix에 따라 종목의 성격이 크게 달라지기 때문에 해외주식 투자 시 참고할 필요가 있다.

## NASDAQ / NYSE / OTC Markets

| Suffix | 의미 | 비고 |
|--------|------|------|
| **A** | Class A 주식 | 의결권 있는 경우가 많음 |
| **B** | Class B 주식 | 의결권 구조가 A와 다름 |
| **C** | 상장 기준 미충족 (Issuer Qualification Exception) | 임시 표시. 상장 기준을 못 맞추고 있으나 잠정 유지 중 |
| **D** | 신규 발행 또는 액면 분할 (New issue / Reverse split) | 임시 표시. 주식 분할·합병 등 기업 구조 변경 중 |
| **E** | SEC 공시 지연 (Delinquent SEC filings) | 임시 표시. 규제 신고를 제때 못 하고 있는 상태 |
| **F** | 외국 기업 발행 주식 (Foreign issue) | |
| **G** | 1종 전환사채 (1st convertible bond) | |
| **H** | 2종 전환사채 (2nd convertible bond) | |
| **I** | 3종 전환사채 (3rd convertible bond) | |
| **J** | 의결권 행사 상황 (Voting share) | 임시 표시. 주주 의결권 이슈 발생 중 |
| **K** | 무의결권 보통주 (Nonvoting common) | |
| **L** | 기타 (Miscellaneous) | 외국 우선주, 우선주 신주 인수권, 추가 유닛 클래스, 워런트 3종, 우선주 6종 등 |
| **M** | 우선주 4종 (4th class preferred) | |
| **N** | 우선주 3종 (3rd class preferred) | |
| **O** | 우선주 2종 (2nd class preferred) | |
| **P** | 우선주 1종 (1st class preferred) | |
| **Q** | 파산 절차 중 (In bankruptcy) | 주의 필요. 상장폐지 가능성 높음 |
| **R** | 신주인수권 (Rights) | |
| **S** | 수익증권 (Shares of beneficial interest) | |
| **T** | 워런트 또는 신주인수권부 주식 (With warrants or rights) | |
| **U** | 유닛 (Units) | |
| **V** | 신규 발행 예정 (Pending issue and distribution) | 주식 분할 등 기업 행동 예정 상태 |
| **W** | 워런트 (Warrants) | 특정 가격에 주식을 살 수 있는 권리. SPAC 관련 종목에서 자주 등장 |
| **X** | 뮤추얼 펀드 (Mutual fund) | |
| **Y** | ADR — 미국 예탁 증권 (American Depository Receipt) | 외국 기업이 미국 시장에 상장할 때 사용하는 형태 |
| **Z** | 기타 (Miscellaneous) | 워런트 2종, 우선주 5종, 스텁, 외국 우선주 신주 인수권, LP 지분 증서 등 |

## TSX / TSXV (캐나다)

캐나다 거래소 기준이라 직접 투자하지 않는다면 참고 수준으로만 알아두면 된다.

| Suffix | 의미 |
|--------|------|
| **A, B** | 주식 클래스 |
| **DB** | 사채 (Debenture) |
| **E** | 배당형 주식 (Equity dividend) |
| **H** | NEX 시장 종목 (상장폐지 유예 종목) |
| **IR** | 분할 납입 증서 (Installment receipts) |
| **NO, NS, NT** | 채권 (Notes) |
| **P** | 자본조달 풀 기업 (Capital Pool Company) |
| **PR** | 우선주 (Preferred) |
| **R** | 청약 증서 (Subscription receipts) |
| **RT** | 신주인수권 (Rights) |
| **S** | 미국 특수 조건 (Special U.S. terms) |
| **U, V** | 미국 달러 펀드 |
| **UN** | 유닛 (Units) |
| **W** | 신주 예정 (When issued) |
| **WT** | 워런트 (Warrants) |

## 자주 마주치는 것들

실제로 해외주식 정보 보다가 많이 보이는 suffix는 이 정도다.

- **W** — 워런트. SPAC 종목에서 자주 등장. 일반 주식과 별개로 거래됨
- **Y** — ADR. 삼성전자 ADR(`SSNLF`), 소니(`SONY`) 같은 외국 기업이 미국 시장에 상장한 형태
- **Q** — 파산 중. 종목명 뒤에 Q 붙어있으면 일단 조심
- **A / B** — 주식 클래스 구분. 버크셔 해서웨이(`BRK.A`, `BRK.B`)가 대표적. 보통 A는 가격이 매우 높고 B는 낮게 쪼개진 형태
- **C / D / E / J** — 임시 표시. 뭔가 기업에 이슈가 있다는 신호이므로 내용 확인 필요
