상장 전 투자서비스 제휴 목적

상장,금투 등등

### 관심 종목 탭 - 관심종목 리스트 조회

- INPUT
    - 고객식별자
    - 기준일자
    - 어떤순서로할지코드
- OUTPUT
    - 리스트카운트
    - 종목리스트
        - 종목코드

### 관심 종목 탭 - 상품 노드 조회

- INPUT
    - 기준일자
    - 종목코드
- OUTPUT
    - 종목명
    - 일반/전문종목
    - 종목코드(6자리)
    - 현재가격
    - 전일대비등락가격(부호포함)
    - 전일대비등락비율(부호포함)
    - 전일최고가
    - 52주최고가
    - 전일최저가
    - 52주최저가
    - 관심종목여부
    - 리스트카운트
    - 90일가격리스트
        - 기준일자
        - 기준가격

밑에 있는 전체 인기종목은 오늘 조회기준을 말함

조회수는 외부에서 주는것만 사용

### 종목 세부

- INPUT
    - 종목코드
    - 기준일자
    - 고객식별자
    - 조회조건(1개월/3개월, 한 번에 줄지 여러번에 줄지에 따라 다름)
- OUTPUT
    - 일반전문종목여부
    - 관심종목여부
    - 기준일자
    - 종목코드
    - 종목명
    - 현재가격(맞음?)
    - 전일종가대비등락률
    - 전일최고가
    - 전일최저가
    - 52주최고가
    - 52주최저가
    - 추정가치
    - 언론에서본기업가치
    - n개월평균가격(1/3개월 기준)
    - n개월최고가격
    - n개월최저가격
    - n개월 리스트카운트
    - 증권플러스URL
    - n개월 가격리스트
        - 기준일자
        - 기준가격

위의 7’ 14200과 밑의 기준가 14700의 차이는 무엇인가

1개월 3개월api를 쪼개줄지 한번에 리스트로 넘길지

### 종목검색 - 검색

- INPUT
    - 기준일자
    - 검색텍스트
- OUTPUT
    - 종목코드 (추가사항 협의중)

### 종목검색 - 최근 종목 키워드 (사용자가 검색해서 눌러서 상세 간 기록 최근 5개 정도?)

- INPUT
    - 기준일시
    - 고객식별자
- OUTPUT
    - 종목코드
    - 종목명

### 종목검색 - 전체 인기 종목

- INPUT
    - 기준일시
- OUTPUT

    - 리스트카운트

        - 종목명
