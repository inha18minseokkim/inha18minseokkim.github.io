```javascript
{
    "detail": {
        "message": "success",
        "code": "SUCCESS",
        "ok": true
    },
    "data": [
        {
            "date": "2024-10-01",
            "symbol": "SHPWQ",
            "entity_name": "Shapeways Holdings",
            "open": 0.0001,
            "high": 0.0001,
            "low": 0.0001,
            "volume": 2,
            "change": 0.0001,
            "value": null,
            "change_percent": 10000.0,
            "trading_halted": false,
            "market_cap": 661.6460000000001,
            "shares_listed": 6616460,
            "type": "stock",
            "close": 0.0001
        }
```


위 주식은 상폐되어서 0.00001로 거래되다가, 상장되어 0.0001로 나스닥에서 거래되기 시작함.
나스닥 시장의 경우 1달러 미만 가격 호가 단위가 0.0001달러이므로 위와같은 문제가 발생.

![](https://prod-files-secure.s3.us-west-2.amazonaws.com/c38aebd7-2834-4fac-b2fc-a2f0c17ce81d/fde633df-7a29-4972-8b08-01447e7af12a/image.png)

결국 상장/비상장 여부를 판단하는 로직에 문제가 있다는 것.
당장 드는 생각엔 산출하는 로직을 전일 종가로 맞춰놓은것 같은데, 전일 종가가 비상장 상태인지 상장 상태인지를 판별하는 로직이 필요할 것 같음. 
해당 건 계속 조치되는지 모니터링 필요.
