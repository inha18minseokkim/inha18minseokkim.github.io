잘 보면 쓸만한 데이터도 많은데, null인 컬럼이 있고 빈문자 컬럼이 모두 있다.

![](attachment:a17ccaec-6e77-488b-8cd0-f30b3cf3f784:image.png)

실제로 존재하는 종목들이고
이렇게 category를 빈 문자를 주는 케이스가 있고

```java
{
  "detail": {
    "message": "success",
    "code": "SUCCESS",
    "ok": true
  },
  "data": {
    "name": "COINSHARES VALKYRIE BITCOIN FUND",
    "country_code": "us",
    "exchange": "NASDAQ",
    "symbol": "BRRR",
    "issuer_name": "CoinShares",
    "category": "",
```
